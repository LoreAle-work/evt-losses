Chapter 8: Multivariate Extremes
================================

Objective:
This chapter studies joint extreme events using multivariate EVT.
The main pair is SPY-QQQ, representing equity-equity joint crash risk.
The contrast pair is SPY-TLT, representing equity-bond joint stress behavior.
The stress-volatility pair is SPY-VIX, representing equity losses and volatility spikes.

Data:
Series: SPY, QQQ, TLT, VIX
Start date: 2002-07-31 
End date: 2026-06-25 
Observations: 6014 

Variable construction:
SPY_extreme = - SPY daily log return
QQQ_extreme = - QQQ daily log return
TLT_extreme = - TLT daily log return
VIX_extreme = VIX daily log return
All variables are oriented so that large positive values are stress events.

Methodological notes:
1. Conditional co-exceedance probabilities are computed at equal marginal quantile levels.
2. They are interpreted as co-exceedance strength, not directional asymmetry.
3. Directional structure is assessed through angular summaries based on W.
4. The implied logistic alpha is derived from empirical chi and is not a full likelihood estimate.
5. Both iid and block bootstrap intervals are reported; block bootstrap is preferred because extremes cluster.
6. Raw structure-variable return levels are interpreted within each pair's units.
7. Frechet-scale structure summaries are better for cross-pair comparison, especially for SPY-VIX.
8. Results are empirical evidence over the observed tail range, not definitive proof of asymptotic dependence.
9. The 99% threshold results are treated as robustness checks because joint counts can be small.
10. The main reported threshold is 97.5%, balancing tail focus and sample size.
